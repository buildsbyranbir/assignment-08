# Herojo Apps (Assignment 08)

## Project Overview
**Herojo Apps** is a frontend project created for Assignment 08. It demonstrates building responsive, accessible UI components and small apps using **HTML**, **Tailwind CSS**, **JavaScript**, and **React**.

This repo includes several example pages/components (e.g., app list, app detail, install/download flow, toast notifications) and is intended to show best practices for component structure, styling with Tailwind, and state management in React.

---

## Tech Stack
- **HTML5**
- **Tailwind CSS** (utility-first styling)
- **JavaScript (ES6+)**
- **React** (Create React App / Vite)

---

## Key Features
- Responsive layout for mobile and desktop
- Reusable React components (AppCard, InstalledAppCard, Toast, Modal, etc.)
- Tailwind-based theming and utility classes
- Accessible markup (semantic HTML, aria attributes)
- Simple routing for multiple pages (Home, App Details, Installed Apps)
